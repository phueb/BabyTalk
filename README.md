A Flask-based micro-service which returns completions of a user's phrase generated by an n-gram model fit on 5M words of child-directed speech.

A demo can be found [here](http://philhuebner.com/projects/baby-talk.html).


### Compatibility

Developed on Ubuntu 16.04 and Python3.7. Designed to be used with FCGI server on hostgator.com
